# Проект автоматизации тестирования SauceDemo

## Описание
Проект демонстрирует использование паттерна PageObject с отчётами Allure для тестирования сайта [saucedemo.com](https://www.saucedemo.com/).

## Технологии
- Python 3.14
- pytest
- Selenium WebDriver
- Allure Framework
- Flake8
